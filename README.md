# ML Learning Journey

This repository documents my hands-on learning journey in Machine Learning and Data Science.  
It contains notebooks and practice projects created while building foundational and intermediate ML skills using publicly available datasets and online learning resources.

The goal of this repository is to demonstrate:
- My learning progression over time
- My understanding of core ML workflows
- My ability to reflect on methodology, limitations, and areas for improvement
