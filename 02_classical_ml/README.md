# Classical Machine Learning Projects

This folder contains applied machine learning projects focused on classical (non–deep learning) techniques such as linear models, tree-based models, and basic pipelines.

The projects here were completed as part of coursework, assignments, and self-directed learning using publicly available datasets. The emphasis is on:
- Problem framing
- Data preprocessing
- Model training and evaluation
- Interpretation of results
- Reflection on limitations and possible improvements

---

## Topics Covered

- Regression and classification problems
- Handling missing values and basic feature engineering
- Model evaluation using appropriate metrics
- Identifying potential data leakage
- Comparing baseline and non-linear models

---

## Notes on Project Scope

- In these projects, the focus was on **analysis and modeling**, not deployment
- Some projects may have methodological weaknesses, reflecting the stage I was in my learning
- Feedback and critique are explicitly welcome

Each project is contained in its own folder with a dedicated README describing:
- The problem
- The dataset
- The approach
- Results
- Limitations and open questions

NOTE: Some notebooks were refactored slightly to remove local paths and proprietary course branding before public sharing.
