# Economic Insights from African Data

## Project Context
This project was completed as a knowledge assessment task for an internship program, focused on exploratory data analysis and insight generation using African economic indicators.

The objective was to demonstrate data handling, visualization, and basic analytical reasoning.

The project notebook can be accessed [here](02_classical_ml/01_economic_insights_africa/WDI_economic_insights_africa.ipynb)

---

## Dataset
- Source: World Bank – World Development Indicators
- Link: [WDI_zipped](https://databank.worldbank.org/AjaxDownload/FileDownloadHandler.ashx?filename=P_884926e9-0b36-4ef9-9787-faa1ef592451.zip&filetype=CSV&language=en&displayfile=P_Data_Extract_From_World_Development_Indicators.zip)
- Countries: Selected African countries
- Years: 2000, 2005, 2010, 2015, 2020
- Features:
  - GDP (USD)
  - Inflation Rate (%)
  - Unemployment Rate (%)
  - Population (millions)

---

## Tasks Performed
- Data loading and inspection using Pandas
- Handling missing values
- Descriptive statistics
- Visualizations:
  - GDP trends for selected countries
  - Inflation trends comparison
  - GDP vs unemployment scatterplot
- Insight summary based on observed trends
